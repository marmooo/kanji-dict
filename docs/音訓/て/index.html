音訓索引「て」行

読み方が「て」から始まる漢字の一覧です。 常用漢字には 57字、JIS第4基準には 577字、Unicode全漢字には 2389字あります。

一覧は例字の欄を参照ください。 学年別漢字配当表、漢字検定配当表、JIS基準、Unicode全漢字を整理して表示します。 このページは Unicode 15.1 に準拠しています。
