石部 (せきぶ)

中国の漢字字典である 康熙字典 では、漢字を 214 の部首に分けています。 そのうち 石部 (せきぶ) に属する漢字をまとめました。 日本では 偏旁 に応じて部首名を以下のように表記・呼称します。
  • 石 (いし)
  • 石 (いしへん)
