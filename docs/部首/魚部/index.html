魚部 (ぎょぶ)

中国の漢字字典である 康熙字典 では、漢字を 214 の部首に分けています。 そのうち 魚部 (ぎょぶ) に属する漢字をまとめました。 日本では 偏旁 に応じて部首名を以下のように表記・呼称します。
  • 魚鱼 (うお さかな)
  • 魚鱼 (うおへん)
Unicode 1.1 に準拠する基本多言語面において、魚部 (ぎょぶ) に属する漢字は 395 字あります。 一覧は例字の欄を参照ください。
