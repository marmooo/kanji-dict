火部 (かぶ)

中国の漢字字典である 康熙字典 では、漢字を 214 の部首に分けています。 そのうち 火部 (かぶ) に属する漢字をまとめました。 日本では 偏旁 に応じて部首名を以下のように表記・呼称します。
  • 火 (ひ)
  • 灬 (れっか れんが)
  • 火 (ひへん)
Unicode 1.1 に準拠する基本多言語面において、火部 (かぶ) に属する漢字は 447 字あります。 一覧は例字の欄を参照ください。
