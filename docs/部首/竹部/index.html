竹部 (ちくぶ)

中国の漢字字典である 康熙字典 では、漢字を 214 の部首に分けています。 そのうち 竹部 (ちくぶ) に属する漢字をまとめました。 日本では 偏旁 に応じて部首名を以下のように表記・呼称します。
  • 竹 (たけ)
  • 竹 (たけかんむり)
Unicode 1.1 に準拠する基本多言語面において、竹部 (ちくぶ) に属する漢字は 378 字あります。 一覧は例字の欄を参照ください。
