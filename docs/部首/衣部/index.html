衣部 (いぶ)

康熙字典 214 部首「衣部」に属する漢字の一覧です。 常用漢字には 21字、JIS第4基準には 131字、Unicode全漢字には 1113字あります。

中国の漢字字典である 康熙字典 では、漢字を 214 の部首に分けています。 そのうち 衣部 (いぶ) に属する漢字をまとめました。 日本では 偏旁 に応じて部首名を以下のように表記・呼称します。

  • 衣 (ころも)
  • 衤 (ころもへん)

一覧は例字の欄を参照ください。 学年別漢字配当表、漢字検定配当表、JIS基準、Unicode全漢字を整理して表示します。 このページは Unicode 15.1 に準拠しています。
